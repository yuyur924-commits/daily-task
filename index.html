Google Sheets 연동 설정
아래 단계를 따라 설정하면 어떤 기기에서든 동일한 데이터를 쓸 수 있어요.
1
Google 시트 만들기
sheets.new 를 새 탭에서 열어 빈 구글 시트를 만드세요.
2
Apps Script 열기
시트 상단 메뉴에서 확장 프로그램 → Apps Script 클릭
3
코드 붙여넣기
기존 내용을 모두 지우고 아래 버튼으로 복사한 코드를 붙여넣은 뒤 💾 저장(Ctrl+S)

✓ 복사됨!
4
배포하기
Apps Script 상단 배포 → 새 배포 → 유형: 웹 앱
다음 사용자로 실행: 본인 / 액세스 권한: 모든 사용자 → 배포
5
웹 앱 URL 복사
배포 완료 후 나오는 웹 앱 URL을 아래에 붙여넣으세요.
웹 앱 URL
URL을 올바르게 입력해주세요.
